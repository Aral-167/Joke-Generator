# Joke Generator
This is a basic joke generator powered by https://official-joke-api.appspot.com/ where you can view your joke history and you can add a joke to your favorites. This joke generator will help you find jokes when you are stuck in a friend group and there is a awkward silence you will just say a joke and will be the most funniest person in the room.
